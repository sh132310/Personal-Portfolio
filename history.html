HTML History

HTML stands for hypertext markup language. This formatting systems shows graphics, text, and audio on a computer monitor. HTML was created by a British scientist named Sir Tim Berners-Lee at nuclear physics laboratory in Switzerland during the 1980's. HTML tags simplify document elements. Web browsers interpret the tags in a layout that is able to adapt to the screen size.

JavaScript History

JavaScript was created in 1995 by a NetScape employee named Brendan Eich. He was given 10 days to create a lightweight scripting language. When JavaScript was originally built, it was called Mocha. It was never intended to become the standard web language. The only reason JavaScript is so popular is because AOL aquired NetScape which then turned over the coding to the Mozillq Fondation.
